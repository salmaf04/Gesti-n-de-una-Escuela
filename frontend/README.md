## Pasos para ejecutar
### Con pnpm (Recomendado) (VPN)

1. **Instalar pnpm:**
   ```sh
   npm i -g pnpm
   ```
2. **Instalar dependencias:**
   ```sh
    pnpm install
    ```
   
3. **Ejecutar el proyecto:**
    ```sh
    pnpm dev
    ```
   
### Con npm (VPN)
1. **Instalar dependencias:**
   ```sh
   npm install
   ```
2. **Ejecutar el proyecto:**
   ```sh
    npm run dev
    ```
### Con yarn (No VPN)
1. **Instalar Yarn:** 
   ```sh
   npm i -g yarn
   ```
2. **Instalar dependencias:**
   ```sh
    yarn install
    ```
3. **Ejecutar el proyecto:**
    ```sh
    yarn dev
    ```
